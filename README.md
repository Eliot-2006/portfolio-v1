# Eliot Boda — Portfolio

Personal portfolio site. Built with Next.js, Tailwind CSS, and TypeScript.

## Live Site

[https://eliotboda.dev/](https://eliotboda.dev/)

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Deployment

Deployed on [Vercel](https://vercel.com). Push to `main` to trigger a production deploy.
